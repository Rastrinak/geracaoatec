* Clonar projeto
    ```sh
    git clone https://github.com/luiscjs/geracaoatec.git
    ```
* copiar ficheiro .env.example para .env
    ```sh
    cp .env.example .env
    ```
* gerar nova key
    ```sh
    php artisan key:generate
    ```
